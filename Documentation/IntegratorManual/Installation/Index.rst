==========================
Installing a Content Block
==========================

Installation
============

Content Blocks can be placed in the `ContentBlocks` folder at any of your extension. The system determines and loads them automatically.

Administration
==============

.. attention::
   You will need to allow the generated database fields, tables (if using inline
   relations) and CType in the backend user group permissions.
