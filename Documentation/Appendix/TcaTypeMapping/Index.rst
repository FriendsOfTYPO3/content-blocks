================
TCA type mapping
================

Content Blocks redefines all Core TCA types. Most of them are the same, but have
a capital letter as the first character. Some are slightly different. Here is an
overview of the mapping. This is also useful to know, if you want to reuse
custom fields.

+----------+---------------+
| TCA      | ContentBlocks |
+----------+---------------+
| category | Category      |
+----------+---------------+
| check    | Checkbox      |
+----------+---------------+
| inline   | Collection    |
+----------+---------------+
| color    | Color         |
+----------+---------------+
| datetime | DateTime      |
+----------+---------------+
| email    | Email         |
+----------+---------------+
| file     | File          |
+----------+---------------+
| flex     | FlexForm      |
+----------+---------------+
| folder   | Folder        |
+----------+---------------+
| language | Language      |
+----------+---------------+
| link     | Link          |
+----------+---------------+
| number   | Number        |
+----------+---------------+
| password | Password      |
+----------+---------------+
| radio    | Radio         |
+----------+---------------+
| group    | Relation      |
+----------+---------------+
| select   | Select        |
+----------+---------------+
| slug     | Slug          |
+----------+---------------+
| input    | Text          |
+----------+---------------+
| text     | Textarea      |
+----------+---------------+
| uuid     | Uuid          |
+----------+---------------+
| json     | Json          |
+----------+---------------+
